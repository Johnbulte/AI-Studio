# AI Studio OS 网页原型设计规格

## 目标

创建一个可运行的 1440×1024 桌面端 AI Studio OS 主界面，忠实复刻 Figma 文件 `pw09dkGSbYLOhRNFzLvuHP` 中节点 `26:773`。交付物以源码为唯一编辑源，适合后续封装到 Tauri。

## 范围

- 深色科技风视觉、中文界面文案与 1440×1024 主画布。
- 左侧导航、顶部栏、创作工作台内容区、创作入口卡片、最近创作与右侧推荐区域。
- 颜色、排版、间距、圆角以 CSS 自定义属性定义。
- 可复用 React 组件：侧栏导航项、操作按钮、内容面板、创作入口卡和最近创作项。
- 静态示例数据与轻量视觉反馈：按钮与卡片焦点/悬停状态；不实现菜单切换或业务逻辑。

## 明确不在范围

- 真实 AI 生成、文件上传、项目保存、登录、网络请求与支付。
- 直接嵌入截图或将截图作为页面背景。
- 不引入 Figma 以外的第三方素材或图标包。

## 技术方案

使用 React、TypeScript、Vite 与纯 CSS。页面以语义化组件树组织；设计令牌置于 `src/styles/tokens.css`，组件均通过 class 与 CSS 变量取色，Figma 导出的本地图像与 SVG 仅用于对应的视觉资产。桌面端迁移时，Web UI 保持不变，文件读写、菜单、窗口控制等能力通过 Tauri command 单独注入。

## 验收标准

1. `npm run build` 成功。
2. 首页在 1440×1024 视口中呈现与 Figma 节点 `26:773` 一致的侧栏、顶部栏与内容层级。
3. 全部可见文案为中文。
4. 不以截图作为页面背景；使用 Figma 提供的图像与 SVG 资源作为各自的界面资产。
5. 设计令牌、组件与布局可在源码层独立修改。
